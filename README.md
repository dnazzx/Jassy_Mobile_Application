## Flutter Node MongoDB Login Registration

`Git Clone`

```
git clone https://github.com/maheshkareeya/flutter-node-mongodb-login-registration.git
```

`Change Directory`

```
cd flutter-node-mongodb-login-registration
```

`Change Directory`

```
cd backend
```

`Run Npm Command`

```
npm install
```

```
node index.js
```

`Change Directory`

```
cd frontend
```

`Run Command`

```
flutter run
```
